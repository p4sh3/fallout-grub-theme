### Installation / update

- **Secure way:**

  - Download install script:

    ```sh
    wget -P /tmp https://github.com/p4sh3/fallout-grub-theme/raw/master/install.sh
    ```

  - Review it at `/tmp/install.sh`

  - Run it:

    ```sh
    bash /tmp/install.sh
    ```

  ```sh
  wget -O - https://github.com/p4sh3/fallout-grub-theme/raw/master/install.sh | bash
  ```
